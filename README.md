## Project Overlord
~~Deseamos la dominación mundial de los scripts en Linux~~

*Modulos del proyecto*

- Crear un Menú para guiar al usuario por los diferentes modulos que se hagan.
- Modulos:

> = Autoeliminación

> = Recuperación

> = Configuración

> = Base de datos

> = Recuperación de archivos

> = Borrado seguro

> = Contraseñas
